The files 'history.csv' and 'settings.csv' are stored in this folder.

The file 'history.csv' stores the current price and time of saving. This allows the user to see how prices change over time. Later on this could be visualized in a graph.

The file 'settings.csv' stores the preferences of the user. The desired price of the item plus the corresponding url. Those are entered using the graphical user interface.
